## Technologies Used

This website was created in stages. First, I went through the Astro blog tutorial, which gets you to a website that looks pretty bad.
I wanted to convert my tutorial result into something usable, so I updated it manually, component by component, while consulting Google Gemini on things I didn't understand.
This was tedious work, and I pretty much only had a blog and home page after a few days of work.
I had so many ideas that I wanted to implement on the site, but I hadn't found time to implement them.

Later, I created a website for my wife, but used Claude Code to generate the website.
I was so pleased with what it produced that I knew I had to use it to update my website with my feature wishlist.
I did that in short order, and what you see now was created with heavy use of Claude.

Though I'm pretty happy with the results, I'm a bit conflicted about AI use.
It's incredible that I'm able to generate these websites in such a short time, but I worry about my engineering mastery.
I'm driven to become a master software engineer, and I have yet to understand how AI fits into my journey.
For now, I'm happy to create things with the assistance of AI, because in my opinion it is better for them to exist with AI than for them to not exist at all.
